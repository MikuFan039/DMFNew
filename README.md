[原工具](https://github.com/danmak-us)由[KPX](https://github.com/CSGA-KPX)开发

"DMF"取自最初的域名 [dm.mikufans.cn] ，没有其他的含义。

请前往本项目的[Releases](https://github.com/MikuFan039/DMFNew/releases)下载本地版
使用方法见命令菜单里的"使用说明"选项。
